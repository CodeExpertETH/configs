# Prettier config

## Usage

### Install

```bash
$ npm i @code-expert/prettier-config -D
# OR
$ yarn add @code-expert/prettier-config -D
```

### Extend this config

in `package.json`

```json
{
    "prettier": "@code-expert/prettier-config"
}
```
